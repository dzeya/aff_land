# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1BM1-BAOziQF05POE_M23j3bQ4RQkr907

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Deploy to Vercel

1. Import this repository into Vercel and choose the **Vite** (Static Site) framework preset.
2. Build command: `npm run build` (already set in `vercel.json`), Output directory: `dist`.
3. Environment variables: add `GEMINI_API_KEY` under Project Settings â†’ Environment Variables (set for Production and Preview).
4. Node.js runtime: Vercel will read `package.json` and use Node `>=20.19.0` to match the Vite plugin requirements.
5. Deploy. The included `vercel.json` uses the static build output and routes all paths to `index.html` so client-side navigation works.
